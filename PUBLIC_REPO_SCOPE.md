# PUBLIC_REPO_SCOPE

## What this repo is

A public-facing SDK and demo surface for studios evaluating embodied NPC behavior.

## What this repo is not

- not the private server runtime
- not the `incarna/` core
- not a disclosure of private policies or thresholds
- not an invitation to inspect confidential release evidence

## Review boundary

If a conversation needs:
- server code
- internal thresholds
- policy tuning
- continuity settings
- confidential manifests

then the conversation has crossed into private review territory.
